# 🌌 Neural Interaction System: Interactive Particle Art

Neural Interaction System waa mashruuc farshaxan dhijitaal ah (Interactive Digital Art) oo ku dhex shaqeeya biraawsarka. Wuxuu abuurayaa shabakad dhibco ah (Constellation Network) oo si firfircoon uga jawaabaya dhaqdhaqaaqa mouse-ka kombiyuutarka iyo taabashada shaashadaha moofaylka (Mobile Touch), isagoo adeegsanaya HTML5 Canvas iyo Tailwind CSS.

---

## ✨ Awoodaha Boggan (Features)

* 🧪 **Dynamic Particle Physics:** Boqol dhibcood (`numberOfParticles = 100`) oo si xor ah shaashadda ugu dhex hulaaqaya, dhalisana midabyo iskugu jira Cyan iyo Blue (`hsl(180-240)`).
* 🕸️ **Constellation Connection:** Marka laba dhibcood ay isku dhowaadaan wax ka yar 150px, waxaa si otomaatig ah u dhex kalaba xadhig iftiimaya oo leh opacity isbedbedesha.
* ⚡ **Mouse & Touch Repulsion:** Marka mouse-ka ama farta la mariyo shaashadda, dhibcuhu si dabiici ah ayay uga fogaadaan (repel) aaggaas iyadoo la isticmaalayo xisaabta `Math.sqrt`.
* 📱 **Mobile Optimized:** Koodhku wuxuu wataa `touch-action: none` si looga horjoogsado moofaylka inuu screen-ka scroll ama zoom ka dhigo, taas oo keenaysa dareen la mid ah in lagu dhex jiro App rami ah.
* 🌀 **Glitch & Pulse Typography:** Magaca bartamaha ku yaal wuxuu leeyahay animation dynamic ah oo weynaanaya, yaraanaya, iskuna beddelaya blur iyo caddaan.

---

## 🛠️ Tiknoolajiyada la isticmaalay (Tech Stack)

* **HTML5 & Canvas API** - Loogu talagalay maamulka xisaabta garaafyada iyo animation-ka dhibcaha.
* **Tailwind CSS** - Naqshadaynta qoraalka overlay-ka ah iyo meelaynta bartamaha (`flex`, `items-center`).
* **Vanilla JavaScript (ES6+)** - Habaynta Object-Oriented Programming (OOP) iyadoo la isticmaalay `class Particle` si loo abuuro loona maamulo dhibic kasta.
* **Custom CSS CSS Animations** - Loogu talagalay saameynta garaaca qoraalka (`@keyframes pulse`).
